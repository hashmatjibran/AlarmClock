About

  1. The Main purpose of this project is to make One understand How to create an Alarm Application using the front end technologies.
  2. I have tried to maintain and keep the functionality of this project very simple and understandable.
  3. I also have used different kinds of technologies like Html , css , bootstrap and JavaScript.
  4. The source code of this project can also be found on my GitHub page which is provided in the footer section of this project.

How To Use?

  1. To create an Alarm Go to Set New Alarm.
  2. A model will Appear After Clicking on the button. There you can easily create a New Alarm.
  3. In Order to view all the Alarms Or edit the Alarms Just click on the View/Edit Alarms button.
  4. You can Also Download the Pdf by clicking on the Download icon here.
